# Welcome to My Blog

This is my first blog post built with **Astro**!

## Why Astro?

Astro is a fantastic framework for building content-focused websites. Here are some reasons why I chose Astro:

1. **Zero JS by default** - Astro sends HTML to the browser, not JavaScript
2. **Islands Architecture** - Only hydrate the components that need interactivity
3. **Content Collections** - Type-safe content management
4. **MDX Support** - Write Markdown with JSX components
